# BRD — 업무자동화 진단 4단계 워크시트

## 1. 비즈니스 목적

Woka의 AI/AX 기업교육에서, 수강생이 **자신의 업무에 AI를 적용할 포인트를 스스로 찾도록 돕는 인터랙티브 워크시트**를 제공한다.

강의 중 실습 도구로 사용하며, 완성된 워크시트를 바탕으로 수강생이 바로 AI(ChatGPT, Claude 등)에게 프롬프트를 전달할 수 있다.

## 2. 비즈니스 배경

- AI 업무자동화 강의에서 가장 큰 허들: "내 업무에서 뭘 자동화할 수 있을까?"
- 기존 강의는 강사가 예시를 보여주는 형태 → 수강생 스스로 찾는 경험이 부족
- 4단계 프레임워크(발견→정의→분해→전환)로 사고 과정을 구조화하면 누구나 자기 업무에 적용 가능

## 3. 핵심 가치

| 이해관계자 | 가치 |
|-----------|------|
| **수강생** | 강의 끝나면 바로 AI에 던질 수 있는 프롬프트를 갖고 나감 |
| **교육 담당자(고객사)** | 수강생이 실제 업무에 적용한 결과를 확인할 수 있음 |
| **Woka(강사)** | 수강생 데이터로 후속 교육 설계 & 성과 리포트 제공 가능 |

## 4. 성공 지표

| 지표 | 목표 |
|------|------|
| 워크시트 완성율 (Step 4 도달) | 80% 이상 |
| 프롬프트 복사/다운로드 비율 | 60% 이상 |
| 강의 만족도 향상 | 기존 대비 +0.3점 (5점 척도) |

## 5. 범위

### In-Scope (구현 완료)
- 4단계 워크시트 웹앱 (Vercel 배포)
- 직무 기반 맞춤 예시 (Gemini 2.5 Flash Lite 연동)
- 프롬프트 자동 생성 + 복사/다운로드
- 이메일 발송 (Gmail SMTP)
- 수강생 작성 데이터 DB 저장 (Supabase)
- 사용자 액션 추적 (복사, 다운로드, 이메일 등)

### Out-of-Scope (향후)
- 관리자 대시보드 (수강생 데이터 시각화)
- 고객사별 브랜딩 커스터마이징
- Resend 도메인 인증 (woka.kr) 후 발신 주소 변경

## 6. 제약 조건

- 무료/저비용 인프라 (Vercel 무료 티어, Supabase 무료 티어, Gemini 무료 티어)
- 수강생 개인정보 최소 수집 (이름, 직무, 이메일만)
- 이메일 주소는 발송 목적으로만 사용, 프론트엔드에 저장하지 않음
- 모바일 우선 (수강생이 휴대폰으로 접속)
- API 키는 Vercel 환경변수에만 저장, GitHub에 미포함
